# Jasmine_Plus

基于大名鼎鼎的Typecho主题Jsamine的二次修改版本，新增了一种展示样式，标题右上角展示，banner图常驻并随机切换功能

其他功能均保持原版，只对修改部分做说明：


## 自定义 Banner 图片使用说明

首页顶部支持显示自定义 Banner 图片，每次访问页面时会随机展示一张图片。

## 使用方法

### 1. 图片存放位置

将 Banner 图片放入以下目录：

```
assets/banner/
```

### 2. 支持的图片格式

- JPG / JPEG
- PNG
- GIF
- WebP
- BMP
- SVG

### 3. 添加图片

直接将图片文件复制到 `assets/banner/` 目录即可，文件名不限。

示例：
```
assets/banner/
├── banner1.jpg
├── banner2.png
├── banner3.webp
└── ...
```

### 4. 显示规则

- 每次页面加载时随机选择一张图片显示
- 如果目录为空或不存在，不显示 Banner 区域
- 图片宽度自适应容器，最大高度 240px
- 图片使用 `object-fit: cover` 保持比例裁剪

### 5. 显示位置

Banner 图片显示在首页顶部，导航栏上方，一直可见。

## 首页文章缩略图设置

### 功能介绍

通过自定义字段 `是否缩略图` 控制首页文章卡片的缩略图显示方式。

### 设置方法

在文章编辑页面，添加自定义字段 `是否缩略图`，可设置以下值：

| 值 | 效果 |
|---|------|
| 留空（不设置） | 不显示任何缩略图 |
| `0` | 不显示任何缩略图 |
| `1` | 提取正文前 3 张图片，以网格形式展示 |
| 图片 URL | 显示指定的单张缩略图 |


### 显示位置

- 带缩略图的文章卡片，图片显示在右侧（桌面端）
- 图片网格样式（`thumbnail=1`）在移动端也会显示

**技术支持：** Kimi
